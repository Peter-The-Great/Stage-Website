# Stage-Website beroeps
# (Dutch)
Een stage website gemaakt voor studenten om een stage aan te geven op een gecentraliseerde website.
Dit is een applicatie gemaakt specifiek voor leerlingen en leraren om stages in te kunnen plannen en beheren.

Project gemaakt door Floris, Dawid, Pjotr, Lars en Jarred

# (English)
An internship website created for students to indicate an internship on a centralized website.
This is an application made specifically for students and teachers to schedule and manage internships.

Project created by Floris, Dawid, Pjotr, Lars and Jarred
